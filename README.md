# 🏏 IPL Data Analytics Dashboard

## 📌 Overview

This project is an interactive IPL Data Analytics Dashboard built using SQL and Power BI. The dashboard analyzes IPL match data to generate insights related to team performance, player statistics, venue trends, and season-wise analysis through visually interactive reports and KPI dashboards.

---

## 🚀 Tools & Technologies Used

* SQL
* MySQL
* Power BI
* Data Cleaning & Transformation

---

## 📊 Dashboard Features

### 🔹 IPL Overview

* Most Successful Teams
* Toss Decision Analysis
* Season-wise Match Trends
* Highest Team Scores
* Interactive KPI Cards

### 🔹 Player Analysis

* Top Run Scorers
* Most Sixes
* Player of the Match Awards
* Player Performance KPIs

### 🔹 Venue Analysis

* Most Used Venues
* Average Runs by Venue
* Highest Scores by Venue
* Venue Performance Insights
